# RateHub

RateHub is a full-stack web application, It allows users to register, log in, browse stores, and submit ratings. The application includes role-based access for Admin, Normal User, and Store Owner.

# Features

- Secure authentication using JWT
- Role-based access control
- Store management
- Store rating system (1–5)
- Admin and Store Owner dashboards
- Search and filter functionality
- Responsive user interface

# Tech Stack

- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: PostgreSQL
- Authentication: JWT, bcrpt

# Project Purpose

This project was built to demonstrate full-stack development skills, including frontend development, backend APIs, database integration, authentication, and role-based authorization.
